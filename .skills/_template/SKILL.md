# Template Skill

## When to use this skill
이 스킬은 새로운 Agent Skill을 생성하기 위한 **기본 템플릿**입니다. 이 폴더를 복사하여 새로운 스킬을 만드세요.

## How to use it
1. 사용자의 요청을 분석하여 필요한 작업 단계를 파악한다.
2. `scripts/` 폴더에 있는 도구가 필요하다면 실행한다.
3. 작업 결과를 사용자에게 간결하게 보고한다.
