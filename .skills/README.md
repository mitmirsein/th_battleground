# 🧩 MS_Dev Agent Skills Library

이 디렉토리는 **Agent Skills** 프로토콜에 따라 정의된 에이전트 스킬들을 모아두는 곳입니다.

## 📂 구조
각 스킬은 하나의 **폴더**로 구성되며, 다음 표준 구조를 따릅니다:

```text
skills/
└── [스킬명]/
    ├── SKILL.md        # [필수] 에이전트 행동 지침 (프롬프트 + 메타데이터)
    ├── scripts/        # [옵션] 실행 코드 (Python, JS 등)
    └── assets/         # [옵션] 참조용 데이터, 템플릿 파일
```

## 🚀 사용법
1. `_template` 폴더를 복사하여 원하는 스킬 이름으로 바꿉니다.
2. `SKILL.md`에 해당 스킬이 무엇을 하는지, 에이전트가 어떻게 행동해야 하는지 적습니다.
3. 필요한 경우 `scripts/`에 파이썬 스크립트를 작성합니다.
4. 에이전트에게 "OOO 스킬을 사용해줘"라고 요청하면, 에이전트가 해당 폴더의 지침을 읽고 수행합니다.
