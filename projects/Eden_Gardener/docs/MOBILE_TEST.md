# 📱 모바일 테스트 가이드

## 1. 같은 Wi-Fi 연결 확인
Mac과 모바일 기기가 **같은 Wi-Fi 네트워크**에 연결되어 있어야 합니다.

## 2. 로컬 서버 접속
모바일 브라우저에서 다음 주소로 접속:

```
http://192.168.0.2:8081
```

또는

```
http://192.168.0.5:8081
```

> ⚠️ 두 IP 중 하나가 작동합니다. 둘 다 시도해보세요.

## 3. iOS - 홈 화면에 추가

1. **Safari**에서 위 주소 접속
2. 하단 **공유 버튼** (📤) 탭
3. **"홈 화면에 추가"** 선택
4. 앱 이름 확인 후 **"추가"** 탭

> 홈 화면에 "에덴 동산지기" 아이콘이 추가됩니다!

## 4. Android - 홈 화면에 추가

1. **Chrome**에서 위 주소 접속
2. 자동 설치 프롬프트가 나타나면 **"설치"** 탭
3. 또는 메뉴 (⋮) → **"홈 화면에 추가"**

## 5. 테스트 체크리스트

| 테스트 항목 | 확인 |
|:---|:---|
| 인트로 화면 로드 | ☐ |
| 아바타 선택 | ☐ |
| 가죽옷 착장 + 서약 | ☐ |
| 겨자씨 심기 | ☐ |
| 일일 묵상 화면 | ☐ |
| 구약/신약 탭 전환 | ☐ |
| 원어 분석 보기 | ☐ |
| 기도 입력 | ☐ |
| 물 주기 | ☐ |
| 동산 뷰 | ☐ |

## 6. 오프라인 테스트

1. 앱을 홈 화면에서 실행
2. **비행기 모드** 켜기
3. 앱이 정상 작동하는지 확인
