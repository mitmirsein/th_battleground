## 🔤 원어 단어 파싱 형식

### 히브리어 (OT)

| 필드 | 설명 | 예시 |
|------|------|------|
| `text` | 원문 단어 | וַיֹּאמֶר |
| `sound` | 한국어 음역 | 바요메르 |
| `lemma` | 기본형 | אָמַר |
| `morph` | 문법 정보 | 접+동 칼 연속과거 3남단 |
| `gloss` | 의미 | 그리고 그가 말씀하셨다 |

### 헬라어 (NT)

| 필드 | 설명 | 예시 |
|------|------|------|
| `text` | 원문 단어 | λόγος |
| `sound` | 한국어 음역 | 로고스 |
| `lemma` | 기본형 | λόγος |
| `morph` | 문법 정보 | 명 주격 남단 |
| `gloss` | 의미 | 말씀 |

## 🤖 AI 묵상 생성 프롬프트

묵상 콘텐츠를 AI로 생성할 때 사용하는 템플릿:

```
당신은 "케리그마 매일 묵상"의 묵상 작가입니다. 

## 필수 규칙

1. **구약과 신약 연관성**: 두 본문의 신학적 연결고리를 명확히 제시
2. **원어 용어 포함**: 히브리어/헬라어 핵심 단어를 음역과 함께 사용
3. **마무리는 위로와 응원**: 마지막 2-3문장은 따뜻한 격려

## 분량 및 문체
- 분량: 400-500자
- 문체: 경건하지만 친근한 어조

## 출력 형식
묵상: [본문]
성찰 질문: [독자를 향한 질문 1개]
```

---

## 🔄 새 월 데이터 추가

1. `data/YYYY-MM.json` 파일 생성
2. 위 스키마를 따라 일자별 데이터 추가
3. GitHub에 푸시 → Netlify 자동 배포

---

## 🛠️ 문제 해결

| 문제 | 해결 |
|------|------|
| 그리스어/히브리어 깨짐 | Noto Sans Greek/Hebrew 폰트 확인 |
| 다크모드에서 안 보임 | CSS `--text` 변수 확인 |
| 오프라인 로딩 실패 | Service Worker 캐시 초기화 |
