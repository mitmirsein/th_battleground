# Task: Academic Prose Polish

주어진 마크다운 리포트를 학술 논문 문체로 개선하세요.

## 핵심 규칙

1. **각주 보존**: `[^1]`, `[^2]` 등 모든 각주 마커를 정확히 보존
2. **헤딩 보존**: `##`, `###` 구조 유지
3. **표 보존**: 마크다운 표 그대로 유지
4. **불릿 → 문단**: 불릿 리스트를 연결된 학술 문장으로 변환
5. **학술 문체**: 간결하고 명료한 문장, 객관적 어조

## 변환 예시

**Before:**
```
루터 신학의 핵심 개념:
- **의인(Justus)**: 그리스도의 의가 전가됨[^1]
- **죄인(Peccator)**: 육신 안에 죄성이 남아 있음
```

**After:**
```
루터 신학의 핵심은 '시물 유스투스 에트 페카토르' 공식에 있다. 이에 따르면, 신자는 그리스도의 의가 전가됨으로써 '의인'으로 인정받는다.[^1] 그러나 동시에 육신 안에는 여전히 죄성이 남아 있어 '죄인'이기도 하다.
```
